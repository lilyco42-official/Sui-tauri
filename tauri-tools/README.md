# Tauri + Vue 3 + Vite + TailwindCSS + DaisyUI 脚手架项目

## 项目简介
该项目是一个基于 Tauri、Vue 3、Vite、TailwindCSS 和 DaisyUI 的脚手架项目。它提供了一个快速搭建前端应用的基础框架，包含了以下功能：
- Tauri 作为前端应用的运行环境，提供了跨平台的能力。
- Vue 3 作为前端框架，提供了响应式的数据绑定和组件化的开发方式。
- Vite 作为前端构建工具，提供了快速的开发体验和高效的打包性能。
- TailwindCSS 作为 CSS 框架，提供了灵活的样式定制和响应式设计能力。
- DaisyUI 作为 UI 组件库，提供了丰富的组件和样式模板，方便快速开发。

## 分支说明
- main: 主分支，包含稳定的功能和修复的 bug。
- template-ui：UI 组件库分支，包含了 DaisyUI 的组件和样式模板。

目前完成了 template-ui 分支的开发，后续会继续完善其他功能。

使用
1. 克隆项目到本地：
```bash
git clone https://gitee.com/bikr/tauri-tools.git
```
2. 进入项目目录：
```bash
cd tools
```
3. 安装依赖：
```bash
npm install
```
4. 启动开发服务器：
```bash
npm run dev
```
5. 打包应用：
```bash
npm run build
```
6. 运行应用：
```bash
npm run tauri dev
```
7. 打包应用：
```bash
npm run tauri build
```
